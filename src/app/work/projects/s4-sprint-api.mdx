## Overview

A compact Spring Boot REST API modeling an aviation domain: **cities**, **airports**, **aircraft**, **passengers**, and **flights**. Entities are returned as JSON with straightforward CRUD operations plus targeted helper endpoints (including aircraft/airport relationships and flight-related lookups).

## Highlights

- **Domain-focused routes**: Separate resource groups for cities, airports, aircraft, passengers, and flights with GET/POST/PUT/DELETE patterns where applicable.
- **Query helpers**: Endpoints such as airports by code or city, aircraft linked to airports (takeoff/landing/combined), and flight-related reads tailored for downstream clients.
- **Local development**: Java 17+, Maven; MySQL or H2 for tests; configuration via `application.properties`.

## Stack

Java 17, Spring Boot, Maven, MySQL / H2.
